# Flappy Bird Clone
**Mustafa Ali — Course End Project**

A Flappy Bird clone built with Python and Pygame.

## How to Play
- Press **Space** to flap
- Fly between the pipes without hitting them
- Each pipe pair you pass = 1 point
- Press **Space** to restart after a game over

## Medals
| Medal    | Score Required |
|----------|----------------|
| Bronze   | 10+            |
| Silver   | 20+            |
| Gold     | 30+            |
| Platinum | 40+            |

## Requirements
- Python 3
- Pygame (`pip install pygame`)

## Run
```
python "Ali_Mustafa - FlappyBird.py"
```
